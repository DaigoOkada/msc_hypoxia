# msc_hypoxia

This repository contains the code associated with the paper:

**Single-cell omics and stochastic modeling reveal hypoxia-driven population-level aging in cultured mesenchymal stromal cells**

## Overview

This codebase supports the analyses presented in the paper, which combines single-cell omics data with stochastic modeling to investigate how hypoxia drives population-level aging in cultured mesenchymal stromal cells (MSCs).
